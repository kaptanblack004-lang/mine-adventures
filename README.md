# 🧠 Word Fight - Kelime Savaşı

Mynet tarzı ultra profesyonel mobil kelime yarışması oyunu! En gelişmiş AI desteği ile online oynayın, zekanızı test edin ve liderlik tablosunda zirveye tırman!

- **Responsive Tasarım** - Tüm mobil cihazlarda mükemmel görünüm
- **Touch Kontrolleri** - Dokunmatik ekranlar için optimize edilmiş
- **Safe Area Desteği** - iPhone X ve sonrası için notch desteği
- **Haptic Feedback** - Titanyum geri bildirim sistemi

- **Stratejik Düşünce** - Durum analizi ve optimal hamleler

## 🚀 Hızlı Başlangıç

### Kurulum
```bash
# Projeyi klonla
git clone https://github.com/kullanici/word-fight.git
cd word-fight

# Geliştirme sunucusunu başlat
python3 -m http.server 8000

# Tarayıcıda aç
# http://localhost:8000
```

### Mobil Kurulum
1. Telefonunuzda tarayıcıda `http://localhost:8000` açın
2. Chrome menüsünden "Ana Ekrana Ekle" seçin
3. Word Fight ikonu ana ekranınıza eklensin
4. Artık her yerden oynayabilirsiniz!

## 🎮 Oyun Nasıl Oynanır?

### Temel Kurallar
1. **Harfleri İncele** - Ekranda verilen harfleri kontrol et
2. **Kelime Oluştur** - Harflerle anlamlı kelimeler oluştur
3. **Hızlı Ol** - Süre daralırken en çok skoru elde et
4. **Strateji Kullan** - Güçlendirmeleri doğru zamanda kullan

### Kontroller
- **Dokunmatik** - Harflere dokunarak seç
- **Swipe** - Sol kaydır: kelime gönder, Sağ kaydır: geri
- **Klavye** - Kelimeyi yazıp Enter'a bas
- **Butonlar** - Karıştır, İpucu, Pas gibi aksiyonlar

### Puanlama Sistemi
- **Temel Puan** - Her harf 10 puan
- **Uzun Kelime Bonusu** - 6+ harf: +20, 8+ harf: +50
- **Kombo Bonusu** - Ardışık bulma: 5x combo = +25 puan
- **Hız Bonusu** - 45+ saniye kala: +15 puan
- **Güçlendirme Bonusları** - 2x puan, zaman durdurma

## 🔧 Geliştirme

### Teknoloji Stack
- **Frontend** - HTML5, CSS3, JavaScript ES6+
- **UI Framework** - Custom CSS with CSS Variables
- **Animation** - CSS Animations + JavaScript
- **PWA** - Service Worker, Web App Manifest
- **Backend** - Node.js, WebSocket, SQLite (isteğe bağlı)

### Geliştirme Ortamı
```bash
# Geliştirme sunucusu
python3 -m http.server 8000

# Live reload (isteğe bağlı)
npm install -g live-server
live-server --port=8000

# Build ve optimizasyon
npm run build
npm run optimize
```
