# Burak Profesyonel Web Sitesi

Bu proje, ultra profesyonel, akıcı ve güvenli bir web sitesi örneğidir. Modern web teknolojileri kullanılarak geliştirilmiştir.

## Özellikler

- **Responsive Tasarım**: Tüm cihazlarda mükemmel görünüm
- **Akıcılık**: Smooth animasyonlar ve geçişler
- **Güvenlik**: XSS koruması, CSP, güvenli header'lar
- **Performans**: Optimize edilmiş kod ve lazy loading
- **Erişilebilirlik**: WCAG standartlarına uygunluk
- **Eğlenceli İçerik**: İnteraktif oyunlar ve animasyonlar
- **Blog**: Güncel içerikler ve yazılar
- **Galeri**: Görsel portfolyo
- **3D Efektler**: Modern animasyonlar ve parçacık sistemleri

## Yeni Profesyonel Özellikler

- **Ultra Yaratıcı Arka Plan**: Çok katmanlı gradient'ler, yıldız pattern'ları, animasyonlu efektler
- **Gelişmiş Güvenlik**: CSP, HSTS, XSS koruması, referrer policy
- **Müzik Player**: İnteraktif müzik çalar (placeholder tracks)
- **Konfigürasyon Yönetimi**: JSON tabanlı ayarlar
- **Logging ve Yedekleme**: Detaylı loglama ve otomatik yedek
- **İstatistikler**: Site analizi komutları
- **Argüman Parsing**: Gelişmiş komut satırı seçenekleri

## Kullanım

### Sunucu Başlatma
```bash
cd python
python main.py server --port 8080 --ssl
```

### İstatistik Görüntüleme
```bash
python main.py stats
```

### Yedek Alma
```bash
python main.py backup
```

## Dosya Yapısı

- `html/index.html`: Ana sayfa
- `css/style.css`: Ultra yaratıcı stiller
- `js/script.js`: İnteraktif özellikler ve oyunlar
- `json/data.json`: Site verileri
- `json/config.json`: Konfigürasyon ayarları
- `python/main.py`: Gelişmiş backend
- `assets/`: Görseller ve medya dosyaları
- `docs/README.md`: Dokümantasyon

## Oyunlar

1. **Tic-Tac-Toe**: Klasik X-O oyunu
2. **Renk Değiştirici**: Rastgele renk animasyonları
3. **Sayı Tahmin**: 1-100 arası sayı tahmin oyunu

## Animasyonlar

- **Parçacık Sistemi**: Canvas tabanlı hareketli parçacıklar
- **3D Küp**: CSS 3D transform ile dönen küp
- **Yazı Glow**: Metin üzerinde ışık efekti

## Teknolojiler

- HTML5
- CSS3 (Flexbox, Grid, Animasyonlar)
- JavaScript ES6+ (Async/Await, DOM Manipulation)
- JSON (Veri yönetimi)
- Python (Backend script'leri)

## Kurulum

1. Projeyi klonlayın veya indirin.
2. HTML dosyasını tarayıcıda açın: `html/index.html`
3. Alternatif olarak Python sunucusunu kullanın:
   ```bash
   cd python
   python main.py server 8000
   ```
   Ardından `http://localhost:8000` adresine gidin.

## Güvenlik Notları

- İçerik Güvenlik Politikası (CSP) uygulanmıştır.
- XSS koruması için input sanitization kullanılır.
- Güvenli HTTP header'ları eklenmiştir.
